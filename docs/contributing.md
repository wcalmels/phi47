# Contributing

Contributions are welcome when they improve correctness, reproducibility, documentation, testing, or scientific clarity.

Before submitting a pull request:

1. create a focused branch;
2. add or update tests;
3. run `py -m pytest -q`;
4. classify any scientific claim correctly;
5. document assumptions and limitations;
6. complete the pull-request template.

See the repository-level `CONTRIBUTING.md` and `CODE_OF_CONDUCT.md`.
